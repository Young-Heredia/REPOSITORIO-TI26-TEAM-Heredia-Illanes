# REPOSITORIO-TI26-TEAM-Heredia-Illanes
Trabajo integrado de investigación, análisis exploratorio, preprocesamiento, entrenamiento y comparación de modelos de aprendizaje automático para **predecir si una oportunidad de venta del CRM será ganada (Won) o perdida (Lost)**, utilizando datos de clientes, productos, equipos comerciales y pipeline de ventas.
